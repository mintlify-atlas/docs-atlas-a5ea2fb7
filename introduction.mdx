<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get the emulator running in minutes</p>
  
  <Steps>
    <Step title="Install Rust toolchain">
      Install Rust and the target for your platform:
      
      ```bash
      # Install Rust
      curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
      
      # Add platform targets
      rustup target add aarch64-linux-android    # Android
      rustup target add aarch64-apple-ios        # iOS
      rustup target add wasm32-unknown-unknown   # Web
      ```
    </Step>
    
    <Step title="Clone the repository">
      Get the source code and build the emulator core:
      
      ```bash
      git clone https://github.com/hunterchen7/ti84ce.git
      cd ti84ce
      cd core && cargo build --release
      ```
    </Step>
    
    <Step title="Build for your platform">
      Use the unified build script to compile for your target:
      
      <CodeGroup>
        ```bash Android
        ./scripts/build.sh android --install
        ```
        
        ```bash iOS
        ./scripts/build.sh ios --sim
        open ios/Calc.xcodeproj
        ```
        
        ```bash Web
        make web-dev
        # Visit http://localhost:5173
        ```
      </CodeGroup>
      
      <Note>For iOS, the build script compiles the backend library. Open Xcode to build and run the app.</Note>
    </Step>
    
    <Step title="Load a ROM and run">
      You need a TI-84 Plus CE ROM file (legally obtained). Import it through the app's file picker, then press Run to start emulation.
      
      <Tip>The emulator supports loading `.8xp` (programs) and `.8xv` (AppVars) files into the calculator's flash archive.</Tip>
    </Step>
  </Steps>
</div>
